
# Расчёт моментов примитивов 

Будем рассчитывать статические моменты, моменты инерции и положение центра тяжести относительно осей координат.

## Общие формулы

### Статический момент
$$S_x = \int_F y dF;$$
$$S_y = \int_F x dF.$$

### Центр тяжести
$$C_x = \frac{S_y}{F};$$
$$C_y = \frac{S_x}{F}.$$

### Моменты инерции

$$J_{xx} = \int_F y^2 dF;$$
$$J_{xy} = \int_F xy dF;$$
$$J_{yy} = \int_F x^2 dF.$$
